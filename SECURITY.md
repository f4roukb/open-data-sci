# Security Policy

OpenDataSci is an autonomous AI agent purpose-built for data science and machine learning. It executes real Python code in a sandboxed environment, so security issues — particularly around sandbox escapes, code execution, or credential handling — are taken seriously.

## Supported versions

Only the latest published release of `open-data-sci` receives security fixes.

## Reporting a vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Report them by email to **b4farouk@gmail.com** with the subject line `[SECURITY] open-data-sci`.

Include:
- A description of the vulnerability and its potential impact.
- Steps to reproduce or a minimal proof-of-concept.
- Any relevant versions or configurations affected.

You will receive an acknowledgement within 48 hours. If the report is confirmed, a fix will be prepared and released as soon as possible, and you will be credited in the release notes unless you prefer otherwise.
